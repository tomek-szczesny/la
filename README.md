# la

A suite of command line tools for processing laser cutter G-Code files.

## General description

The aim of this project is to create a complete toolchain for creating G-Code files for laser cutters.

Pure command line interface allows for full design automation and maximum efficiency. la commands are intended for use in scripts and batch jobs.

All la commands accepting G-Code parse it, ignoring content that is not G-Code or is not supported by the parser. Only codes G0, G1, G90 and G91 are currently supported as input, as this is a complete set of commands describing the actual content of the job. The G-Code output is always expressed in abolute coordinates (G90). 

The final G-Code file for laser cutters often requires additional G-Codes at the beginning and/or end of the job file. Those are to be added by the user after processing with la tools. See `examples/process.sh` for an example working with Creality Falkon cutters. This example can be easily adapted to other machines by examining their example gcode files.

## Commands

| Command                 | Arguments                       | stdin  | stdout       | Description                                                                                                                                                                                    |
| ----------------------- | ------------------------------- | ------ | ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| la_strip                | none                            | G-Code | G-Code       | Parses input file and rewrites its contents to the output. In effect, it strips input file from all unrecognized content, such as comments or fringe op codes.                                 |
| la_dedup                | none                            | G-Code | G-Code       | Deduplicates cuts. This involves removing exact overlaps and resolving partial overlaps.                                                                                                       |
| la_bounds               | none                            | G-Code | comment line | Generates a comment line for G-Code file, containing job's XY bounds. The format conforms to that generated by LightBurn. Some laser cutters use this information for framing.                 |
| la_pos                  | -x [number]<br/>-y [number]     | G-Code | G-Code       | Translates the entire content so its bounding box's lower left corner has user-specified X,Y coordinates. Arguments default to 0,0 which aligns the input with cutter's x and y axes.          |
| la_rot                  | -a [number]<br/>-minx<br/>-miny | G-Code | G-Code       | Rotates anti-clockwise the entire content either by a given angle (-a), or to minimize its bounding box dimension along X or Y axis. Content is rotated around the center of its bounding box. |
| la_svg2g                | none                            | SVG    | G-Code       | Converts SVG data into G-Code.                                                                                                                                                                 |
| la_g2svg                | none                            | G-Code | SVG          | Plots G-Code in SVG format.                                                                                                                                                                    |
| la_g2pdf                | none                            | G-Code | PDF          | Plots G-Code in PDF format.                                                                                                                                                                    |
| la_opt                  | none                            | G-Code | G-Code       | Attempts to rearrange cuts to minimize idle travel (G0).                                                                                                                                       |
